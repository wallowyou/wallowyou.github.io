webpack5 搭建vue3环境 #

使用pnpm管理安装依赖

安装webpack相关依赖 #

pnpm install webpack webpack-cli webpack-dev-server webpack-merge cross-env -wD

增加webpack配置文件 #

在根目录与src文件夹同级别的地方增加一个build文件夹，下面分别增加三个配置文件webpack.common.js,webpack.dev.js,webpack.prod.js

增加各种webpack loader #

  1. ts-looader 处理ts文件，此处先用ts-loader，之后使用esbuild测试两者速度
  2. asset/resource 处理静态文件加载images图像，字体
  3. style-loader css-loader加载css文件
  4. vue-loader 处理.vue文件

常用的webpack插件 #

  1. html-webpack-plugin
  2. mini-css-extract-plugin
  3. copy-webpack-plugin
